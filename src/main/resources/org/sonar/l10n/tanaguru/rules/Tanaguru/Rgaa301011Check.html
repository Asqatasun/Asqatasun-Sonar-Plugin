L'attribut alt permet de définir l'alternative textuelle d'une image. Elle est utilisée lorsque l'image actuelle ne peut être restituée pour une des causes suivantes :

  • L'image ne peut pas être trouvée (404)
  • L'utilisateur utilise un assistance technique pour lire la page
  • La connexion internet est lente, et le chargement des images est ralenti
  • Le chargement des images est désactivé, par exemple pour réduire la consommation des données sur terminaux mobile

Example de code non conforme

<img src="foo.png" />       <!-- Non conforme -->

Example de code conforme

<img src="foo.png" alt="Information véhiculée par foo.png" />       <!-- Lorsque foo.png porte une information -->
<img src="foo.png" alt="" />                                        <!-- Lorsque foo.png est une image purement décorative -->
