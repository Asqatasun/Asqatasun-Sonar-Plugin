La déclaration <!DOCTYPE> indique au navigateur quelle version Html est utilisée dans la page, et donc les règles de syntaxe à appliquer dans cette page. Cette déclaration doit toujours précéder la balise <html>

Example de code non conforme

<html>            <!-- Non-Compliant -->
...
</html>

Example de code conforme

<!DOCTYPE html>
<html>            <!-- Compliant -->
...
</html>
