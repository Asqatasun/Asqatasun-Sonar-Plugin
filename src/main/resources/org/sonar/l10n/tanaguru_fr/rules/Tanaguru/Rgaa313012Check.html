L'utilisation de <meta http-equiv="refresh"> est découragée par le World Wide Web Consortium (W3C). Si l'utilisateur utilise le bouton 'Back', certains navigateurs vont revenir sur la page redirigée, plutot que sur la page attendue par l'utilisateur, celle qui précède la page redirigée.

Pour rafraichir une page, l'utilisation d'Ajax est une possibilité, pour mettre à jour uniquement les éléments qui ont besoin d'être rafraichis plutot que de rafraichir la page entière. Attention toutefois, dans ce cas, à utiliser Aria pour Pour rediriger vers une autre page, privilégier les modes de redirection serveur avec les codes de retour HTTP 301 'Moved Permanently' ou 302 'Found'

Exemple de code non conforme

<head>
  <meta http-equiv="refresh" content="5">   <!-- Non-Conforme -->
  <meta name="description" content="...">
</head>

Exemple de code conforme

<head>
  <meta name="description" content="...">
</head>

<head>
  <meta http-equiv="refresh" content="0">   <!-- Conforme -->
  <meta name="description" content="...">
</head>
